Table - organization of information or data in rows and columns.

tr - a table row

th - a header cell in a table

td - a cell in a table

thead - a header content in a table

tfoot - a footer content in a table

caption - a table caption

Colspan - merges multiple columns in the HTML table.

Rowspan - merges multiple rows in the HTML table.

Cells spacing - space between a table cell's border and the content in it.

Padding - space around the element's content inside of any defined borders.

To define table borders using CSS, set a type of border around the table, like thick, thin, collapsible, dotted, double or dashed.

To control border widths using CSS, set the number value to define how thick the border around the table should be.
